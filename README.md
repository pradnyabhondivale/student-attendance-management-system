## AttendEase : Track. Analyze. Excel.

AttendEase is a comprehensive web-based Student Attendance Management System built with Flask and SQLite. 
It features real-time attendance tracking, role-based access for Admin, Faculty, and Students, detailed analytics with graphs and reports, and easy data export in Excel/CSV formats.

## Features
- Role-based authentication and authorization for Admin, Faculty, and Students  
- Real-time attendance marking and percentage calculation  
- Student and class-wise attendance dashboards with graphical insights  
- Notice board for timely communication between faculty/admin and students  
- Export attendance records to CSV/Excel files  
- User-friendly responsive interface


## Usage
- **Admin:** Manage users, classes, and view system-wide attendance statistics
- **Faculty:** Mark attendance, manage subjects and classes, and view reports
- **Student:** View personal attendance records and notices


## Technologies Used
- Python 3, Flask Framework
- SQLite database
- Frontend: HTML, CSS, JavaScript
- Data visualization: Chart.js
